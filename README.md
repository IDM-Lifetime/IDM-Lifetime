## Download & install (archive delivery)
1. Click **Download 5x Now** — download starts (`.zip`).
2. Extract to `C:\IDM-Lifetime-Pack`.
3. Run `Install_IDMLifetime.exe`.
4. Launch IDM → tools auto-activate.
> Pro tip: Use 16 connections for max speed.
